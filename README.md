# SpeechBot

Um **AIBot** que consegue capturar áudio, processar e responder de forma auditiva.
Sistema **Multiplataforma**.

---

## ⚙️ Configuração

### Crie um arquivo `.env`

Em seu arquivo `.env`, coloque as seguintes variáveis:

- **Gemini** = **_Seu token de autenticação do Gemini_**
- **local** = **_Caminho completo para o modelo que está em models_**
